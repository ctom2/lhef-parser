# Les Houches Event File Analyser Tool

Simple LHEF parser written in Python that can be used without ROOT. Created for [SuperChic](https://superchic.hepforge.org) Monte Carlo generator output file (in Les Houches Evet File format) analysis. Its main purpose is to extract information about the particles present in the following productions:
* γγ → γγ
* γγ → µµ
* γγ → ee
* Axion-like particle production
